# 《钢雨前线》内容设计圣经

日期：2026-04-30

本文档扩展《钢雨前线》的角色、武器、敌人、关卡、美术和长期内容方向。目标是让后续策划、美术、程序都能按同一套标准生产内容。

## 1. 设计原则

### 1.1 爽快优先

玩家每次开火都要有重量：

- 枪口火焰明显。
- 敌人中弹有硬直、火花、血雾或装甲碎片。
- 爆炸有屏幕震动、低频音效、烟尘和残骸。
- 强武器要有夸张反馈，不能只提高数字。

### 1.2 美丽但可读

画面要优美，但战斗信息必须清楚：

- 玩家轮廓始终明亮。
- 敌方子弹统一用红橙高亮。
- 可互动道具用蓝白闪光。
- 危险地形用红色预警或明显动画。
- 背景细节不能和敌方子弹抢视觉。

### 1.3 多样但不乱

每一关最多同时引入 1-2 个新机制。敌人、地形、武器都要有明确用途。

内容变化节奏：

- 每 30 秒有小刺激。
- 每 2-3 分钟有新敌人或地形变化。
- 每 5-7 分钟有小高潮。
- 每关结尾有大 Boss 或大型事件。

## 2. 角色设计扩展

### 2.1 艾拉 Aila

视觉关键词：

- 短外套、轻装护甲、红色围巾、蓝灰主色。
- 动作干净利落，像特种部队队长。
- 表情冷静，受伤时仍保持克制。

战斗特点：

- 均衡型。
- 翻滚性能好。
- 标记敌人，提高队伍输出。

专属被动：战场直觉。

- 敌方狙击和导弹预警提前 0.25 秒出现。
- 救援 NPC 时奖励额外弹药。

专属技能：战术标记。

- 范围内敌人被标记 6 秒。
- 被标记敌人受到 20% 额外伤害。
- Boss 弱点高亮。

角色台词风格：

- 少话。
- 精准。
- 轻微冷幽默。

例：

- “火力点确认。”
- “别站在爆炸旁边，这不是建议。”
- “推进。”

### 2.2 赫维 Harvey

视觉关键词：

- 高大、重甲、机械臂、弹链、黄色警示条纹。
- 动作有重量，落地有尘土。

战斗特点：

- 生命高。
- 移动慢。
- 正面抗压。
- 重武器稳定。

专属被动：重装骨架。

- 正面受到的小型子弹伤害降低 30%。
- 机枪和火箭筒后坐力降低。

专属技能：防爆盾推进。

- 展开盾牌 4 秒。
- 正面小型子弹无效。
- 可缓慢移动。
- 盾牌结束时释放一次震波。

性格：

- 豪爽。
- 爱嘲讽敌人。
- 对队友很可靠。

### 2.3 燕 Yan

视觉关键词：

- 轻甲、短发、红黑配色、双枪、腿部推进器。
- 动作夸张，空中姿态漂亮。

战斗特点：

- 高机动。
- 空中冲刺。
- 近战爆发。
- 生命低。

专属被动：连击升温。

- 连续击杀提高移动速度和近战伤害。
- 受击后连击归零。

专属技能：影步斩。

- 瞬移到最近敌人背后。
- 造成高伤害。
- 若击杀敌人，返还 50% 冷却。

性格：

- 张扬。
- 爱挑衅。
- 危险越大越兴奋。

### 2.4 林熙 Lin Xi

视觉关键词：

- 工程背包、护目镜、机械无人机、绿色/白色主色。
- 动作不花哨，但工具展开很精致。

战斗特点：

- 控场。
- 炮塔。
- 地雷。
- 修理载具。

专属被动：现场改装。

- 补给箱有概率额外掉落武器弹药。
- 载具生命恢复量提高。

专属技能：蜂群协议。

- 召唤 3 架无人机，持续 8 秒。
- 自动攻击最近敌人。
- 对机械单位伤害提高。

性格：

- 理性。
- 说话直接。
- 对机器比对人更温柔。

### 2.5 诺亚 Noah

视觉关键词：

- 黑白长风衣、能量眼罩、长枪、冷色光效。
- 动作安静，射击瞬间极有爆发。

战斗特点：

- 远程爆发。
- 蓄力穿透。
- 标记弱点。
- 容错低。

专属被动：弱点解析。

- 命中敌人弱点返还技能能量。
- 对 Boss 装甲破坏效率提高。

专属技能：裂隙光束。

- 蓄力 0.8 秒后发射直线激光。
- 穿透敌人。
- 对大型单位多段命中。

性格：

- 沉默。
- 神秘。
- 偶尔说出很准确的判断。

## 3. 武器设计扩展

### 3.1 武器形态定位

每把武器三个形态必须分别对应：

- 基础输出。
- 特定敌人解法。
- 高消耗爆发或控场。

不要做三个只差数值的形态。

### 3.2 突击步枪

用途：默认可靠武器。

形态：

- 连射：普通敌人、移动战。
- 穿甲：盾兵、装甲兵、薄墙后敌人。
- 榴弹：掩体后敌人、小范围群体。

升级方向：

- 连射稳定性提升。
- 穿甲击中第二个敌人时伤害不衰减。
- 榴弹爆炸后留下短烟雾，遮挡敌方普通兵视线。

### 3.3 霰弹枪

用途：近距离高风险高回报。

形态：

- 散射：近距离清杂。
- 独头弹：中距离单体爆发。
- 燃烧：地面封锁。

升级方向：

- 散射命中 4 发以上触发击退。
- 独头弹打弱点必定硬直。
- 燃烧地面持续时间增加。

### 3.4 火箭筒

用途：重型爆发，弹药珍贵。

形态：

- 直射：Boss 和装甲单位。
- 分裂：群体敌人。
- 追踪：无人机和高速目标。

升级方向：

- 直射命中弱点返还少量弹药概率。
- 分裂子弹数量增加。
- 追踪锁定速度提高。

### 3.5 能量枪

用途：科技感、控制和穿透。

形态：

- 激光：持续直线伤害。
- 电链：多个敌人弹跳。
- 护盾脉冲：近距离击退敌人和子弹。

缺点：

- 过热系统。
- 连续使用会降低射速。

### 3.6 火焰喷射器

用途：近距离压制和区域控制。

形态：

- 持续火焰：贴脸高伤。
- 火球：中距离爆裂。
- 地火：制造燃烧区域。

特殊互动：

- 点燃油桶。
- 烧毁藤蔓。
- 对部分变异体有恐惧效果。

## 4. 敌人阵营

### 4.1 黑礁军团

人类军阀部队。第一关主敌。

单位：

- 步枪兵
- 手雷兵
- 盾兵
- 狙击手
- 喷火兵
- 迫击炮兵
- 通讯兵
- 装甲车驾驶员

视觉：

- 旧式军装。
- 黄色识别臂章。
- 拼装装甲。
- 武器粗糙但火力足。

战斗风格：

- 多人协同。
- 火力压制。
- 掩体和炮击。

### 4.2 企业军“赫利俄斯”

第二阶段敌人。更高级、更冷酷。

单位：

- 激光步兵
- 护盾无人机
- 医疗无人机
- 重装机兵
- 自爆机器人
- 浮游炮平台

视觉：

- 白色装甲。
- 金色灯带。
- 干净、昂贵、没有人味。

战斗风格：

- 高精度。
- 护盾。
- 无人机协同。

### 4.3 遗迹变异体

来自古代能源污染。

单位：

- 爬行者
- 爆囊怪
- 巨臂兽
- 飞膜虫
- 晶化寄生体
- 遗迹守卫

视觉：

- 生物组织和晶体混合。
- 蓝紫能量纹路。
- 动画不规则。

战斗风格：

- 非线性移动。
- 突袭。
- 自爆。
- 地形穿越。

## 5. Boss 设计扩展

### 5.1 第一关 Boss：巴别-01 巨门战争机

定位：

- 第一关终点。
- 战争机器压迫感。
- 测试玩家基础操作。

阶段 1：外部炮台。

- 左右机炮交替扫射。
- 导弹落点预警。
- 玩家破坏两个炮台。

阶段 2：核心门打开。

- 中央激光横扫。
- 两侧刷敌。
- 玩家需要在清敌和打核心间切换。

阶段 3：过载坍塌。

- 地面平台断裂。
- Boss 核心外露。
- 玩家在移动平台上完成最后输出。

美术表现：

- 机炮被毁后冒黑烟。
- 装甲板脱落。
- 核心发光越来越不稳定。
- 结尾爆炸引出遗迹能量。

### 5.2 第二关 Boss：赫利俄斯无人指挥舰

场景：

- 地下实验室升降井。

机制：

- Boss 悬浮。
- 召唤无人机。
- 激光网限制移动。
- 玩家可打掉护盾发生器。

重点：

- 让玩家熟悉垂直移动和空中单位。

### 5.3 第三关 Boss：晶化巨臂兽

场景：

- 遗迹深井。

机制：

- 大手砸地。
- 墙面爬行。
- 召唤小型变异体。
- 弱点在晶化肩部和胸口核心。

重点：

- 生物 Boss。
- 节奏和机械 Boss 完全不同。

## 6. 第一关详细节奏

### 6.1 总时长

目标：22-28 分钟。

检查点密度：

- 普通段每 3-4 分钟。
- Boss 战阶段间给隐形检查点。

### 6.2 关卡情绪曲线

0-3 分钟：登陆冲击。

- 炮火。
- 海浪。
- 敌兵密集但弱。
- 玩家快速上手。

3-8 分钟：村落推进。

- 节奏稍降。
- 加入救援和隐藏路线。
- 第一次给玩家探索空间。

8-13 分钟：雨林壕沟。

- 压力上升。
- 炮击、泥地、盾兵组合。
- 小 Boss 出现。

13-18 分钟：遗迹转折。

- 美术风格变化。
- 变异体出现。
- 神秘感增强。

18-22 分钟：铁轨载具。

- 速度变快。
- 敌人从前后夹击。
- 给玩家爽快火力。

22-28 分钟：堡垒 Boss。

- 最高压。
- 多阶段。
- 通关后爆炸和剧情钩子。

## 8. 美术生产标准

### 8.1 角色

每名角色需要：

- 正面立绘。
- 半身 UI 头像。
- 小头像。
- 游戏内骨骼模型。
- 14 套基础动作。
- 1 套专属技能动作。
- 3-5 条受击/攻击表情。

基础动作：

- idle
- run
- jump
- fall
- land
- crouch
- prone
- shoot
- shoot up
- shoot diagonal
- reload
- dodge
- melee
- hurt
- death

### 8.2 敌人

普通敌人最低动作：

- idle
- walk
- aim
- shoot
- hurt
- death

特殊敌人额外动作：

- windup
- special attack
- stun
- armor break

### 8.3 场景

每个地形主题需要：

- 远景层。
- 中景层。
- 主碰撞层。
- 前景遮挡层。
- 可破坏物。
- 互动机关。
- 天气/粒子。
- 主题灯光色板。

### 8.4 特效

特效分级：

- S 级：Boss 爆炸、剧情爆炸、角色大招。
- A 级：火箭、榴弹、载具爆炸。
- B 级：普通枪口、命中、弹壳。
- C 级：灰尘、脚步、水花。

## 9. 音频标准

### 9.1 武器音效

每把武器至少 5 层：

- 枪口瞬态。
- 机械动作。
- 低频冲击。
- 尾音。
- 环境反射。

### 9.2 角色语音

频率：

- 开场一句。
- 捡强武器一句。
- Boss 入场一句。
- 低血量一句。
- 通关一句。

不能每次开火都说话。

### 9.3 动态音乐

每关音乐分层：

- 探索层。
- 战斗鼓点。
- 高压层。
- Boss 层。
- 胜利尾奏。

## 10. 长期成长系统

第一版不做复杂养成，但保留接口。

可解锁：

- 角色皮肤。
- 武器外观。
- 武器熟练度徽章。
- 角色档案。
- 关卡挑战评级。

不建议早期加入：

- 数值刷装。
- 随机词条。
- 大量资源货币。
- 抽卡。

原因：

- 本项目核心竞争力是手感、美术、关卡。
- 过早复杂养成会拖慢垂直切片。

## 11. 垂直切片内容边界

垂直切片只验证：

- 艾拉手感。
- 三把武器九形态。
- 八种敌人。
- 第一关前 8-10 分钟。
- 一个小 Boss。
- HUD 和检查点。

不验证：

- 五角色完整制作。
- 全 28 分钟第一关。
- 完整美术资产。
- 多人合作。
- 主机适配。
- 完整剧情演出。

验收口径：

- 玩起来爽。
- 看起来有潜力。
- 武器切换有意义。
- 敌人组合有变化。
- 第一关节奏能支撑继续扩展。
