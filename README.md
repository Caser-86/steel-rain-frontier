# Steel Rain Frontier

《钢雨前线》 is a 2D side-scrolling run-and-gun game project inspired by classic arcade action shooters.

Current focus: Unity vertical slice with one playable character, multi-form weapons, varied enemies, checkpoints, HUD, and the first 8-10 minutes of Level 01.
